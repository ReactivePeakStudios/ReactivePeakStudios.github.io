HeroZitus

The Discord Multi-Bot

HeroZitus is a Discord Multi-Bot designed to improve your server experience. Key HeroZitus features include:

Calculating a mathematical problem.

Tracking a country or worldwide COVID-19 cases.

Displaying the number of days since the COVID-19 crisis started.

Displaying the number of members that are in the current server.

Mentioning someone random in the server.

All of the current commands are:

$calc

Calculate a mathematical problem.

$covid

Track a country or worldwide COVID-19 cases.

$days

Displays the amount of days since the COVID-19 crisis started.

$help

Displays all commands and their functions.

$impostor

Sends the impostor screen from Among Us with the name placed after the command.

$members

Displays the amount of members that are in the current server.

$ping

How many milliseconds it takes for your message to be recieved by Discord.

$servers

Displays the amount of servers that the bot is in.

$someone

Mentions someone random in the server.

$calc

Calculate a mathematical problem.
